Agent Setup Guide

Install the tracking agent on any laptop you want to monitor. The agent checks in when the laptop connects to the internet and sends heartbeats every 60 seconds.

  1. Copy agent/agent.py to the target laptop.
  2. Install dependencies:
    pip install requests psutil
  3. Run the agent pointing at this server:
    python agent.py --server http://YOUR_SERVER_IP:5000
  4. Auto-start on boot (Windows — save as a .bat and add to Task Scheduler):
    @echo off start /min python C:\tracker\agent.py --server http://YOUR_SERVER:5000
  5. Auto-start on boot (macOS — add to LaunchAgents):
    python3 /opt/tracker/agent.py --server http://YOUR_SERVER:5000 &
  6. Auto-start on boot (Linux — add to crontab):
    @reboot python3 /opt/tracker/agent.py --server http://YOUR_SERVER:5000 &
  7. Once the agent runs, it will appear in the Devices tab automatically.
